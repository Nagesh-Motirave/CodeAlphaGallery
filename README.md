📸 Gallery Web Project
📌 Project Overview

This project is a responsive image gallery website built using HTML, CSS, and JavaScript. It provides a smooth and interactive user experience with modern UI features such as animations, transitions, and a lightbox view for images.

🌐 Live Demo

👉 View Project Here:
🔗 https://nagesh-motirave.github.io/CodeAlphaGallery/

🚀 Features
🖼️ Image Gallery Layout – Organized and visually appealing gallery design
📱 Responsive Design – Works on mobile, tablet, and desktop devices
🎬 Smooth Transitions & Animations – Enhances user experience
🔍 Lightbox View – Click on images to view in full screen
⏭️ Next/Previous Navigation – Easily browse images
⚡ Fast & Lightweight – Built with pure HTML, CSS, and JavaScript

🛠️ Technologies Used
HTML5 – Structure of the webpage
CSS3 – Styling, layout, and animations
JavaScript – Navigation, lightbox functionality, and interactivity

📂 Project Structure
Gallery/
│── index.html
│── style.css
│── script.js
│── images/

🎯 How to Use
Clone the repository:
git clone https://github.com/Nagesh-Motirave/CodeAlphaGallery.git

Open the project folder:
cd CodeAlphaGallery
Open index.html in your browser

💡 Key Functionalities
Click on any image to open it in lightbox mode
Use Next/Previous buttons to navigate through images
Enjoy smooth animations and transitions
Fully responsive layout adapts to screen size

📸 Future Improvements
Add image categories or filters

📝 Conclusion
This project demonstrates how to build a modern, responsive gallery using core web technologies. It focuses on user experience, smooth interaction, and clean design.
